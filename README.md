# GoldLedger

Monthly gold income/expense tracker for World of Warcraft.

Трекер доходов и расходов голды.

## Features

- **On Hand** — current gold balance
- **Today / Session / Month** — income, expense, and net for each period
- **Source Breakdown** — detailed income/expense per source (Vendor, Repair, AH, Mail, Quest, Loot, Trade) with period filter (Today / Week / Month / All)
- **Daily Chart** — bar chart with period switching (7d / 30d / All) and hover tooltips
- **Goal** — set a gold goal with progress bar and ETA
- **Recent Transactions** — scrollable log with source filter buttons
- **Multi-character** — summary across all characters
- **CSV Export** — export all data for spreadsheets
- **Minimap Button** — draggable, click to toggle
- **Localized** — English (enUS) and Russian (ruRU)

## Commands

| Command | Description |
|---------|-------------|
| `/gl` | Toggle main window |
| `/gl reset` | Reset character data |
| `/gl goal <amount>` | Set gold goal |
| `/gl goal clear` | Clear goal |
| `/gl help` | Show help |

## Installation

1. Download from [CurseForge](https://www.curseforge.com/wow/addons/goldledger) or [GitHub Releases](https://github.com/tumour/GoldLedger/releases)
2. Extract `GoldLedger` folder into `World of Warcraft/_retail_/Interface/AddOns/`
3. Restart WoW or `/reload`
